# W3C RDF Dataset Canonicalization test vectors

Test-only fixtures copied verbatim from the W3C rdf-canon test suite
(https://w3c.github.io/rdf-canon/tests/, test IDs test003c, test020c, test028c, test044c,
test060c, test072c). Each `<name>-in.nq` input must canonicalize byte-for-byte to
`<name>-rdfc10.nq` under RDFC-1.0 with SHA-256.

These files are not part of any shipped artifact.
